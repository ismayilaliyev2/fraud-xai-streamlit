Built an end-to-end Fraud Detection with Explainable AI project using Python, XGBoost, SHAP, FastAPI, Streamlit, Docker, and GitHub Actions. The system generates synthetic transaction data, trains a fraud risk model, evaluates business-focused metrics such as precision, recall, F1-score, ROC-AUC, PR-AUC and latency, and provides a Streamlit dashboard where users can test transactions and view SHAP explanations for each prediction.
